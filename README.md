# Курсовая работа — Библиотека для работы с графами (C++17)

Данный проект реализует библиотеку для работы с неориентированными графами на основе **матрицы смежности**.  
Программа включает:

- класс `Graph` (добавление/удаление вершин)
- операции со связями между вершинами
- загрузку и сохранение графа в файл
- интерактивное консольное меню
- модульные тесты на **GoogleTest**
- сборку через **CMake**

---

## Возможности

### 1. Добавление вершины
При добавлении вершины автоматически увеличивается матрица смежности.

### 2. Удаление вершины
Удаляется строка и столбец матрицы, соответствующие вершине.

### 3. Соединение вершин
Метод:
connect_vertex(int index1, int index2);
устанавливает связь между вершинами по их индексам.

### 4. Разрыв связи
disconnect_vertex(int index1, int index2);
удаляет связь между двумя вершинами.

### 5. Сохранение графа в файл
Формат:

первая строка — значения вершин

далее — матрица смежности N x N

### 6. Загрузка графа из файла
Загрузка поддерживает тот же формат.

### 7. Интерактивное меню
Работает в консоли.
Позволяет:
- добавлять вершины
- удалять
- соединять
- разрывать связи
- сохранять
- загружать
- очищать граф
- просматривать текущий граф

Меню обновляется после каждого действия.

### 8. Тестирование через GoogleTest
В проект включён набор модульных тестов:
- добавление вершин
- удаление
- соединение/разрыв
- сохранение/загрузка
- очистка графа
- оператор <<

Сборка проекта
Требования:
- C++17
- CMake 3.14+
- g++ / clang / MSVC

Команды:
```
mkdir build
cd build
cmake ..
cmake --build .
```
Запуск программы
После сборки:

```
./kursovaya       # Linux/macOS
kursovaya.exe     # Windows
```
Программа запустит интерактивное меню вида:

```
1 - Добавить вершину
2 - Удалить вершину
3 - Соединить две вершины
4 - Разорвать связь
5 - Показать граф
6 - Сохранить граф
7 - Загрузить граф
8 - Очистить граф
0 - Выход
```
Запуск тестов
В папке build:

bash
```ctest```

или:

bash
```
./kursovaya_tests
```

первая строка — значения вершин

остальные строки — матрица смежности
